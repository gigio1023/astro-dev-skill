# Install astro-dev for Gemini CLI

## Preferred

```bash
npx skills add gigio1023/astro-dev-skill@astro-dev --agent gemini-cli
```

## Manual install

```bash
git clone https://github.com/gigio1023/astro-dev-skill.git ~/.gemini/astro-dev-skill
mkdir -p ~/.gemini/skills
cp -R ~/.gemini/astro-dev-skill/skills/astro-dev ~/.gemini/skills/
```

Restart Gemini CLI after copying.
