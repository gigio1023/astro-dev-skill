# Install astro-dev for Cursor

## Preferred

```bash
npx skills add gigio1023/astro-dev-skill@astro-dev --agent cursor
```

## Manual install

```bash
git clone https://github.com/gigio1023/astro-dev-skill.git ~/.cursor/astro-dev-skill
mkdir -p ~/.cursor/skills
cp -R ~/.cursor/astro-dev-skill/skills/astro-dev ~/.cursor/skills/
```

Restart Cursor after copying.
